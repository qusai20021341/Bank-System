# Bank System 💰

A robust **Banking Management System** built in **C++** using strong **Object-Oriented Programming (OOP)** principles.

This console-based application simulates a full banking environment with separate interfaces for **Administrators/Users** and **Clients**, complete with secure authentication, transaction processing, and data persistence.

## ✨ Features

### 🔐 User & Authentication Management
- Role-based access control (Admin / User)
- Secure login and registration system
- User management (Add, Update, Delete, List users)
- Login activity logging

### 👥 Client Management
- Add new clients
- Update client information
- Delete clients
- Find clients
- View all clients
- View total balances report

### 💳 Transactions
- **Deposit** money
- **Withdraw** money
- **Transfer** money between accounts
- Full transaction history and transfer logs

### 💱 Currency Exchange
- View list of currencies
- Find specific currency
- Currency calculator
- Update currency exchange rates

### 📊 Additional Features
- Clean screen-based UI with navigation menus
- Input validation and error handling
- Persistent data storage using text files
- Audit logging for transfers and login activity
- Date and utility helper classes

## 🛠️ Technologies & Architecture

- **Language**: C++ (Visual Studio Project)
- **Paradigm**: Object-Oriented Programming (OOP)
- **Data Storage**: Text files (`.txt`)
- **Design Pattern**: Screen-based architecture with dedicated classes for each feature
